# Dashboard Olaclick


## Setup

Make sure to install the dependencies:

```bash
# npm
npm install

```

## Development Server

Start the development server on `http://localhost:4006`:

```bash
# npm
npm run dev

```

## Production

Build the application for production:

```bash
# npm
npm run generate

```

## The tools:

Preprocesadores:
```
PUG(HTML)
Stylus(CSS)
```
