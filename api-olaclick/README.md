# api Olaclick


## Setup

Make sure to install the dependencies:

```bash
# npm
npm install

```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

```

## Production

Build the application for production:

```bash
# npm
npm run generate

```

## The tools:

Core:
```
Express
MongoDB
Mongoose
Node v20.18.0
```
